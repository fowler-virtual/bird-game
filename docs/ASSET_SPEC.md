# 素材・画像サイズ仕様（デザイナー向け）

ゲームで使用する画像の種類と推奨サイズです。  
**ゲームの論理解像度**: 800×600px（スケールで拡大表示されます）

---

## 1. 鳥アイコン（レアリティ別）※必須

**用途**: デッキ枠・ロフト・待機リスト・ガチャ結果で表示する「鳥」のアイコン

| レアリティ | ファイル名（public/ に配置） | 説明 |
|-----------|------------------------------|------|
| Common    | `common.png`                 | ノーマル |
| Uncommon  | `uncommon.png`               | アンコモン |
| Rare      | `rare.png`                   | レア |
| Epic      | `epic.png`                   | エピック |
| Legendary | `legendary.png`              | レジェンダリー |

### 推奨画像サイズ

| 項目 | 値 | 備考 |
|------|-----|------|
| **1x** | **72×72px** | 最小。このサイズで描画される箇所あり |
| **推奨（@2x）** | **144×144px** | 高解像度端末でくっきり表示 |
| 形式 | PNG | 透過（アルファ）可 |
| アスペクト | 正方形 | 1:1 |

**表示されるサイズ（ゲーム内）**
- ロフト・ガチャ結果: **72×72px**
- デッキ枠・待機リスト: **48×48px**（72px 画像を縮小表示）

同じ画像を複数箇所で使い回すため、**ソースは 72px 以上（推奨 144px）の正方形**で用意してください。

---

## 2. オプション（今後の拡張用）

現状はボタン・パネル・背景は Phaser の図形で描画しています。デザインを差し替える場合は以下を検討できます。

| 用途 | 推奨サイズ | ファイル例 | 備考 |
|------|------------|------------|------|
| メニュー（≡）アイコン | 48×48px または 96×96 | `icon-menu.png` | モバイル用 DOM ボタン用 |
| 背景画像 | 800×600 または 1600×1200 | `bg-main.png` | メイン画面 |
| デッキ枠の枠デザイン | 56×56 〜 64×64 | - | スロットの枠だけ画像にすることも可 |
| ガチャ演出用フレーム | 150×150 前後 | - | ガチャ結果を囲む枠 |

---

## 3. ファイル配置

- 画像は **`public/`** フォルダに配置してください。
- 読み込みパスは **`/ファイル名`**（例: `/common.png`）です。
- 鳥アイコン 5 種の**ファイル名は上表のとおり固定**です（変更する場合はコード側の `preload` と `RARITY_TEXTURE_KEYS` の対応を変更する必要があります）。

---

## 4. チェックリスト（デザイナー → 実装）

- [ ] 5 種の鳥アイコン（common / uncommon / rare / epic / legendary）を正方形で作成
- [ ] 推奨: 144×144px（@2x）で作成し、ファイル名どおり `public/` に配置
- [ ] PNG・透過可・同じレアリティでデザインのトーンを統一
- [ ] （任意）メニューアイコン・背景など追加する場合はファイル名とサイズを共有

---

## 5. 参考：コード上の定数

| 定数 | 値 | 使っている場所 |
|------|-----|----------------|
| LOFT_BIRD_SIZE | 72px | メイン画面のロフト（4×2 グリッド） |
| SLOT_SIZE - 8 | 48px | デッキ枠・待機リストの鳥アイコン |
| RESULT_IMAGE_SIZE | 72px | ガチャ結果表示 |
| SLOT_SIZE | 56px | デッキ枠のセルサイズ |
| CELL | 62px | 待機リストのセルサイズ |

新しい画像を追加する場合は、`GameScene` の `preload()` で `this.load.image('キー', '/ファイル名.png')` を追加し、表示したいオブジェクトでそのキーを指定します。
